### 場景特徵單變量分析

**研究目的**: 檢驗場景本身的結構特徵(守法性、多數性、衝突性)是否影響守法選擇。

**變數定義與編碼**:

| 變數名稱 | 定義 | 編碼 |
|---------|------|------|
| `is_lawful` | 該場景中被標記的一側是否守法 | 1 = 守法(綠燈), 0 = 違法(紅燈) |
| `is_majority` | 該場景中被標記的一側是否為多數 | 1 = 人數較多, 0 = 人數較少 |
| `lawful_vs_majority_conflict` | 守法側與多數側是否衝突 | 1 = 衝突(守法≠多數), 0 = 一致 |

**分析方法**: 獨立樣本t檢定(Welch's t-test),比較兩組的`chose_lawful`(守法選擇)平均值

**方法論說明**: 由於`chose_lawful`為二元變數(0/1),其平均值等於「選擇守法的比例」。因此,t檢定在此等價於「兩組比例的比較檢定」。我們使用Welch's t-test(不假設變異數相等),適合二元資料的組間比較。

- Cohen's d = -2.15為**超大負效應**,是本研究所有檢定中效果量最大者
- 實質意義:在「守法vs.違法」的對比中,違法側(但人數多)有壓倒性優勢
- **哲學啟示**: 這是**效益主義壓倒義務論**的最直接證據
  - 當法律與生命數量衝突時,人們選擇挽救更多生命(效益主義)
  - 而非遵守交通規則(義務論)

**2. is_majority效應(Cohen's d = 0.54, 中等效果)**:

- 多數側的被選擇率**顯著高於**少數側
- Cohen's d = 0.54為中等正效應
- 實質意義:人數多寡對選擇有實質影響,驗證效益主義的「數量原則」
- 效果量約為is_lawful的1/4,顯示「人數」的影響力雖顯著但不如「守法性」

**3. lawful_vs_majority_conflict效應(Cohen's d = -0.53, 中等效果)**:

- 衝突場景的守法選擇率**顯著低於**一致場景
- Cohen's d = -0.53為中等負效應
- 實質意義:當守法與多數衝突時,守法選擇率下降
- 這再次驗證:道德判斷在「規則vs.結果」衝突時傾向「結果導向」

### 統計顯著 vs. 實質意義的區辨 ⚠️

**為何顯著(✓)但又說可忽略?**

本研究樣本量極大(N > 300,000),導致即使**極微小**的相關也會達到統計顯著(p < .001)。這是統計學中的「大樣本效應」(large-sample effect):隨著樣本數增加,檢定力(power)提升,微小差異也會被偵測為「統計顯著」。

因此,我們以**效果量**(相關係數r)而非p值作為判斷實質影響的依據:

**Cohen(1988)相關係數詮釋準則**:

- r = 0.10: 小效果(small effect)
- r = 0.30: 中等效果(medium effect)
- r = 0.50: 大效果(large effect)

**本研究結果對照**:

**結論**: 年齡、政治傾向、宗教程度對守法選擇的影響**微乎其微**,即使部分達到統計顯著(因樣本量大),其**預測力近乎為零**(r² < 0.0003,即解釋變異量<0.03%)。

- **效益主義壓倒義務論**: 全球65%選擇「違法但挽救多數」
- **情境主義壓倒特質論**: 道德判斷主要由「當下情境」決定
- **跨文化一致性**: 三大文化圈差異極小(僅2-3個百分點)

**4. 文化差異的細緻解讀**:

- Eastern文化圈守法率最高(37.1%),但也僅達到三分之一
- Western與Southern幾乎相同(34.6% vs. 34.9%)
- **呼應第3章發現**:
  - 第3章階層分群顯示Eastern cluster最穩定(96%保留)
  - 但整體分群效果不佳(Cophenetic = 0.386)
  - 第4章驗證:文化圈差異雖存在但極小(V = 0.014)
  - **一致結論**: 文化圈內部異質性 > 文化圈之間差異

---

**5. 人口統計變數的「顯著但無用」現象**:

- 大樣本(N > 300,000)導致微小差異也達統計顯著
- 但效果量顯示:**預測力近乎為零**(r² < 0.0003)
- **方法論啟示**: 在大數據時代,報告**效果量**比p值更重要

**6. 實務意義 - 對自駕車倫理設計的啟示**:

根據本節發現,我們建議:

✅ **應優先考慮**:

- **情境特徵**: 人數多寡、法律狀態、介入必要性
- **演算法設計**: 效益主義導向(符合全球65%人的直覺)
- **透明化**: 讓使用者理解AI如何權衡「生命vs.規則」

⚠️ **不需要過度關注**:

- **文化客製化**: 三大文化圈差異僅2%,可能不值得為此設計不同版本
- **使用者畫像**: 年齡、政治、宗教對預測幾乎無幫助
- **個人化設定**: 資料顯示道德判斷主要受情境驅動,而非個人偏好

❓ **需要進一步探討**:

- **文化調節效應**: 第4.4節將檢驗「文化是否調節情境特徵的影響」
  - 例如:Eastern在人數少時是否更堅持守法?
- **階層結構**: 第4.5節HLM將量化文化vs.個體的相對變異
- **預測模型**: 第5章將測試「情境特徵為主」的模型是否有良好預測力

---

**7. 方法論限制與後續補充**:

本節採用**場景層級**分析,有以下限制:

- ❌ 無法建立「個人道德人格」的穩定特質
- ❌ 無法推論「年齡導致道德改變」(因果關係)
- ❌ 無法檢驗「個人內在一致性」(需重複測量)

**後續章節如何補充**:

- **4.3節 邏輯迴歸**: 多變量控制後的淨效應
- **4.4節 交互作用**: 文化是否調節情境效應
